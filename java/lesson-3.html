 Java Lesson 3 - Printing To The Terminal

Your first "Hello, World!"

In the java file you created last time, add

System.out.println("Hello, World!")

in the space between the two center curly braces. Your program should look like

public class Main {
  public static void main(String args[]) {
    System.out.println("Hello, World!")
  }
}

Compile and run this program, and you should see

Hello, World!

right under where you ran it.


Compiling your first program

To compile a java program, we use the javac command that comes with the JDK we installed in the last lesson.

Open vscode's terminal by pressing ctrl+`, and run

javac Main.java

This will create a new file called Main.class which is the compiled version of your program.

If the file was not created, or if the command outputted an error message, then something is wrong with your code.


If you named your file something other than Main, just replace Main.java with the name of your file. For example, if my program was called PotatoMachine.java, I would need to run

javac PotatoMachine.java

Next up: running the program you just compiled!


Running your program

Running your program is very simple, the only thing to do is run

java Main

in the vscode terminal.


If you called your file something else, for example PotatoMachine, you'd instead run

java PotatoMachine

If you're wondering why nothing happens when you run it, it's because we haven't programmed it to do anything yet. We'll do that in the next lesson.


If, for some reason, something did happen, like an error message, make sure that you run java Main (or your own file name) with no typos, and no .java or .class at the end.


In the next lesson, we'll make your program output something when it's run.
