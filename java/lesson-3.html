No Semicolons Left Behind!

 Java Lesson 3 - Printing To The Terminal

Your first "Hello, World!"

In the java file you created last time, add

System.out.println("Hello, World!")

in the space between the two center curly braces. Your program should look like

public class Main {
  public static void main(String args[]) {
    System.out.println("Hello, World!")
  }
}

Compile and run this program, and you should see

Hello, World!

right under where you ran it.


Why is there an error?

In java, every line needs to be ended with a semicolon, to separate one line of code to the other so the compiler knows how to compile our program.

So in our code, we need to add a semicolon after our println statement. Your code should look like

public class Main {
  public static void main(String args[]) {
    System.out.println("Hello, World!");
  }
}

If you're wondering why there aren't any semicolons after the 4 other lines, that's because they all end with a curly brace.

For now, just remember that in java, every line has to end with either a semicolon or a curly brace.


Re-compile and run it and now you should actually see some text get printed.


Changing the text

If you want to print something other than

Hello, World!

then you just need to change the text in between the quotation marks to be whatever you want. For example, this

System.out.println("Potato Machine");

will print

Potato Machine

Make sure to remember the quotes, your code won't work without them!
