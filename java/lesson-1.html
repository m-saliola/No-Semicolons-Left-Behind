 Java Lesson 1 - Getting Started

Installing java

First things first, we need to install a Java Development Kit (JDK), so that we can actually code in java. Many different companies have made their own JDKs, but we'll be using Oracle's version.


Start by going to Oracle's JDK download page and download the correct version for your OS. Download whatever version of java is featured at the top of the website, although the version doesn't matter too much.


Once the installer has been downloaded, run it and install the JDK like any other app.

To verify that java is installed correctly, run

java --version

In your terminal. It should output the version of java that you installed.


Installing a text editor

The next step is installing a text editor, so that we have a way to actually write code. I recommend using vscode, but you can follow this guide with any other app if you want.

Click on the download button in the center of the vscode website, and then download it like any other app.


Once you're in the app, you can optionally install plugins that will help you code in java.

Click on the little icon with 4 blocks on top of each other, and search for the Java extension made by Oracle Corporation.


Once you have JDK and your text editor, you're ready to start coding!
