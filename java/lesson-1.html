 Java Lesson 1- Getting Started

Installing a compiler

Before we can even start writing in C++, we'll need to install a compiler. We'll be using g++.


On mac, it's as simple as running

brew install gcc

in your terminal to install g++.


For windows, it's a bit more complicated. There is no standalone g++ you can install, instead you need to install a dev environment such as msys2 or tdm-gcc. In this tutorial, we'll be using msys2.

Go to the msys2 website, and follow their instructions on installation.


Once you've installed g++, run

g++ --version

to confirm that it's correctly set up.

If it recognizes the command, all's well and good.


Installing a text editor

The next step is installing a text editor, so that we have a way to actually write code. I recommend using vscode, but you can follow this guide with any other app if you want.

Click on the download button in the center of the vscode website, and then download it like any other app.


Once you're in the app, you can optionally install plugins that will help you code in c++.

Click on the little icon with 4 blocks on top of each other, and search for the C/C++ extension made by Microsoft.


Once you have your compiler and your text editor, you're ready to start coding!
