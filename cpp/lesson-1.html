Installing a compiler

Before we can even start writing in C++, we'll need to install a compiler. We'll be using g++.


On mac, it's as simple as running

brew install gcc

in your terminal to install g++.


For windows, it's a bit more complicated. There is no standalone g++ you can install, instead you need to install a dev environment such as msys2 or tdm-gcc. In this tutorial, we'll be using msys2.

Go to the msys2 website, and follow their instructions on installation.


Once you've installed g++, run

g++ --version

to confirm that it's correctly set up.

If it recognizes the command, all's well and good.
