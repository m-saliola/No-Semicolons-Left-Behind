No Semicolons Left Behind!

 C++ Lesson 3 - Printing To The Terminal

Your first "Hello, World!"

In the c++ file you created last time, add

std::cout << "Hello, World!" << std::endl

in the space between the two curly braces. Your program should look like

int main() {
  std::cout << "Hello, World!" << std::endl
}

You also need to include iostream, which gives c++ the ability to print. Think of it as telling the program to learn how to print, before actually attempting to write to the terminal.

The finished program should be

#include <iostream>

int main() {
  std::cout << "Hello, World!" << std::endl
}

Compile and run this program, and you should see

Hello, World!

right under where you ran it.


Why is there an error?

In c++, every statement needs to be ended with a semicolon, to separate one line of code to the other so the compiler knows how to compile our program.

So in our code, we need to add a semicolon after our std::cout statement. Your code should look like

#include <iostream>

int main() {
  std::cout << "Hello, World!" << std::endl;
}

If you're wondering why there aren't any semicolons after the other lines, that's because they either end with curly braces or include a header, such as iostream, and lines like those don't need semicolons.

For now, just remember that in c++, every line has to end with either a semicolon or a curly brace, unless they include something.


Re-compile and run it and now you should actually see some text get printed.


Changing the text

If you want to print something other than

Hello, World!

then you just need to change the text in between the quotation marks to be whatever you want. For example, this

System.out.println("Potato Machine");

will print

Potato Machine

Make sure to remember the quotes, your code won't work without them!
