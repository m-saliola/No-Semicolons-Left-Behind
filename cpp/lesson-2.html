 C++ Lesson 2 - Writing Your First C++ Program

There's a little setup!

Create a new file in vscode by clicking File/New File. Save it with ctrl+s (command+s if you're on mac) and name it main.cpp. You don't have to name it main, that part can be whatever you want, but the .cpp at the end is essential!


Every program written in c++ has this same block of code that is necessary for the program to work.

int main() {
  return 0;
}

Don't worry, you don't need to understand what this does just yet. Just know that every program needs these few lines of code.

Paste it into the file you created earlier, then we'll compile it!


Compiling your first program

To compile a c++ program, we use the g++ compiler we installed in the last lesson.

Open vscode's terminal by pressing ctrl+`, and run

g++ main.cpp -o main

This will create a new file called main (main.exe if you're on windows) which is the compiled version of your program.

If the file was not created, or if the command outputted an error message, then something is wrong with your code.


If you named your file something other than main, just replace 'main' with the name of your file. For example, if I called my program potato_machine.cpp, I would need to run

g++ potato_machine.cpp -o potato_machine
